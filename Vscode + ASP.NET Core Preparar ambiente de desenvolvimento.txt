1. Crie um diretório para seu projeto com a linha de comando a seguir:
mkdir ~/projeto-teste

2. Navegue até este diretório, caso você ainda não esteja nele:
cd ~/projeto-teste

3. Inicie um projeto default:
dotnet new console
